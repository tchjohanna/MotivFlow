💡 Le saviez-vous / Neurosciences

Le cerveau associe la répétition à la récompense dopaminergique. Un petit pas quotidien envoie un signal positif qui renforce progressivement la motivation intrinsèque. Après 21 jours, le comportement commence à s'automatiser. Après 66 jours, il devient une habitude ancrée.

⚠️
Compatibilité des notifications Les notifications web fonctionnent uniquement si l'onglet est ouvert (ou l'app installée). Voici comment maximiser la fiabilité :
  • Android + Chrome : installe l'app via "Ajouter à l'écran d'accueil" → notifications fiables en arrière-plan.
  • iPhone / iPad (iOS 16.4+) : installe via Safari → "Sur l'écran d'accueil" → ouvre depuis l'icône, puis active les rappels.
  • Desktop Chrome/Edge : fonctionne si le navigateur reste ouvert.
  • Safari desktop / Firefox mobile : notifications web non supportées — utilise l'app installée si possible.
